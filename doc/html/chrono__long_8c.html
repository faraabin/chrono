◆ CHECK_RUN_

#define CHECK_RUN_ ( ret)    if(!me->_run){return (ret);}

Checks whether the chrono object is in the run state or not.

Note
If the chrono object is not running, this macro forces the API to return the values specified by "ret".

◆ ELAPSED_

#define ELAPSED_ ( a,
b )   (((a) >= (b)) ? ((a) - (b)) : (ULLONG_MAX - ((b) - (a))))

Returns the time length between "a" and "b".

Note
Chronologically, "a" must be a time tick after "b", but the value of "a" could be less than "b" (e.g., when tick overflows).
