◆ fChrono_GetContinuousTickMs()

uint64_t fChrono_GetContinuousTickMs ( void )

Returns the continuous tick value converted to milliseconds since calling fChrono_Init().

Attention
Always remember to call this function before tick generator overflows. For example if tick generator reaches its topValue in 71 minutes, make sure fChrono_GetContinuousTickMs() is called once in this interval.
Return values
continuousTickTimeTime length since calling fChrono_Init() in milliseconds

◆ fChrono_GetTickToNsCoef()

uint32_t fChrono_GetTickToNsCoef ( void )

Returns the coefficient for converting one tick to nanoseconds.

Note
This coefficient is used to convert ticks (which are unitless) to time lengths with proper units.
The chrono module cannot measure time lengths smaller than this multiplier.
Other multipliers (tick to milliseconds and tick to seconds) are calculated based on this coefficient.
Return values
tickToNsCoefTick-to-nanoseconds coefficient
tickToNsCoefTick-to-nanoseconds coefficient

bool_t _isTimeout

Holds the timeout state of the chrono object. This parameter is private and is set by the chrono API. Users must not change its value.

◆ _run

bool_t _run

Holds the RUN state of the chrono object. When this parameter is false, the chrono is in stop mode, and time measurement functions will return 0. This parameter is private and is set by the chrono API. Users must not change its value.
